# Character System — Profiles, Custom Characters & Community Marketplace

---

## Overview

User can select a "character profile" that constrains the narrative engine to a specific franchise's soundtrack and emotional mapping. The LLM scores the user's day / game / RP AS IF they were that character.

This works across ALL three products:

- **Life:** Your daily commute scored as James Bond
- **Gaming:** Your Valorant session scored as Doom Slayer
- **RP:** Your Genshin conversation scored as Zhongli

---

## How It Works

When a roleplay profile is active:

1. The LLM system prompt gets CHARACTER-SPECIFIC instructions appended
2. Song selection pool narrows to the character's franchise soundtrack
3. Narrative classification adds character-specific scene types
4. Crossfade behavior changes per character (Bond = smooth/composed, Rick = jarring/chaotic, Alastor = vinyl-crackle effect)

---

## The Character Select UX

Home screen: subtle "TODAY I AM..." button at the top. Tap it → **CHARACTER SELECT SCREEN** appears.

Styled like a fighting game / RPG class selection roster. Each character has:

- Silhouette/aesthetic motif (legally distinct but evocative)
- Short tagline capturing the vibe
- 5-second audio preview of the profile's sound
- "Select" button

**Default mode** (no roleplay): Full personalized Underscore profile from user's complete library.
**Roleplay mode:** OVERRIDES with character-specific constraints.
**Blend Mode (Premium):** Assign different characters to different time blocks. Morning: Shelby. Commute: Bond. Work: Saul. Evening: Rick. Frankenstein your own protagonist.

## Custom Character System

### Overview

Users type ANY character name. The LLM (Gemini 3 Flash) auto-generates a complete Underscore profile. This includes emotional architecture, franchise soundtrack mapping, scene-to-context translation, conditional triggers, and transition style.

**This is the REAL product.** The pre-built roster is a showcase. The custom system is infinite.

### Key Architecture Requirements

The custom character system MUST understand:

1. **Character identity, not just franchise.** Reinhard != Subaru within Re:Zero. Same soundtrack library, completely different emotional architecture and track selection.

2. **Character-specific soundtrack filtering.** Reinhard's scenes use the heroic/knightly compositions. Subaru's suffering themes do NOT apply to Reinhard. The LLM must correctly filter within a franchise.

3. **Conditional triggers based on character relationships.** Gojo's soundtrack shifts when people he CARES about are in danger, not when HE is. "Your Battle is My Battle" only plays when detecting that someone the user cares about is struggling — not during the user's own conflicts.

4. **Narrative arc awareness.** Robb Stark mode WILL play ominous Lannister music at social gatherings because the character's story makes that narratively appropriate. Choosing a doomed protagonist means the app scores the doom truthfully.

### More Custom Character Examples

**Gojo Satoru** — Conditional triggers are the key feature:
- Default: "Lost in Paradise" energy (casual, sunglasses-on cool)
- Tense moments: "Delirious" / Hollow Purple intensity
- When OTHERS are in danger: "Your Battle is My Battle" — triggers on detected stakes for side characters, NOT for Gojo himself. The emotional weight is in watching someone you care about fight, not fighting yourself.

**Robb Stark** — Narrative arc awareness:
- Default: Northern drums, Celtic strings, honor and glory
- Social gatherings: "The Rains of Castamere" creeps in. The app knows how Robb's story ends at social events. This is a FEATURE. Choosing Robb means accepting Robb's story, including the ominous parts.

**Tywin Lannister** — Single-theme variations:
- Full orchestral Castamere for dominance moments
- Solo cello Castamere for quiet planning
- Piano Castamere for evening reflection
- Nearly-subliminal hummed Castamere for dark satisfaction
- One theme, seventeen arrangements, each for a different context

## Community Marketplace

### Features

- Share custom profiles publicly with one tap
- Rate/review community profiles (star rating + text reviews)
- "Most Popular" and "Staff Picks" categories
- Profile remix — fork someone's profile, adjust it, share your version
- Franchise collections — "All Game of Thrones characters," "Complete Genshin Impact roster"
- Search by character name, franchise, mood, archetype

### Content Moderation

Community profiles need basic moderation. The LLM should refuse to generate profiles that map to harmful real-world behaviors. Standard community moderation practices apply. User reporting system for inappropriate profiles.
